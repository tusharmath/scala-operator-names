# scala-operator-names

List of all the popular Scala operators, their names and their typical usage.

| **Operator** | **Common Name** |
| ------------ | --------------- |
| `*>`         | Ice Cream       |
| `<*`         | Reverse Cream   |
| `<>`         | Diamond         |
| `++`         | Fence           |
| `+++`        | Wide Fence      |
| `>=>`        | Fish            |

That's all!
